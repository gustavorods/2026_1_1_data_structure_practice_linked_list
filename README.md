# Data Structure Practice – Linked List (Countries and Borders)

Java exercise focused on practicing linked list structures, object relationships, and traversal operations using a simple geographic scenario.

## Description

This project implements a simple data structure where each country stores its borders using a linked list.

- **Note:** You can complete this exercise individually by following the `exercise.md` file.

The system is composed of two main classes:

- **Pais** – represents a country and stores the list of its borders

- **NoFronteira** – represents a node in the linked list containing a neighboring country

Using this structure, the project demonstrates how to:

- Create and use a **custom linked list**
- Insert elements into the list
- Traverse the list using a loop
- Remove elements from the list
- Search for elements in the structure
- Represent **relationships between objects**

The main program creates some European countries and defines their borders, demonstrating how the linked list is used to manage these relationships.

The goal is to practice **fundamental data structure concepts in Java**, especially how linked lists work internally.

## Installation

To get started, clone the repository and navigate to the project directory:

```bash
git clone https://github.com/gustavorods/2026_1_1_data_structure_practice_linked_list.git <project_name>
cd <project_name>
```

Then compile and run the Java files.

Example:

```bash
javac *.java
java Principal
```

## Usage

The program demonstrates the following operations:

1. Creating countries
2. Adding borders between countries
3. Listing borders of a country
4. Checking if a country borders another

Example output:

```bash
Fronteiras do Alemanha:
Holanda
Belgica
Franca
```
